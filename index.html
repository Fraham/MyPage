My Second Year Project - EasyLiving

For my second group project we were left to come up with our own ideas to produce any type system we wanted. We finalised on a room monitoring system which collects enivormental information from different areas in the house. As a group we decided to use PHP on the backend server and jQuery for frontend development. We felt these were the best choices available as they both lots of online tutorials and resources. We also used the Bootstrap framework which allowed us to build a website with one codebase while providing access from desktops down to mobile with its ability of using the correct element widths for the screen size. The idea behind the project was to provide people the ability to monitor a set of rooms from any device with an internet connections so we built the website with the focus of mobile first then working on larger screen sizes. The initial site I was able to retrieve the data from our MySQL database then package the data using PHP into a JSON object which is sent to the page where I was able to set up client rendering of the data.
